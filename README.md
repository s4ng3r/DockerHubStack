# DockerHubStack
Pre-configured docker compose files for local development.

+ Databases: MongoDB, SQL Server, Postgres, Cassandra
+ Cache: Redis, Memcached
+ Message broker: Kafka, RabbitMQ
+ Secrets: Vault
+ Test: SonarQube
+ Monitoring: Grafana
+ Logs: Logstash
